# 📝 To-Do App

A simple and user-friendly **To-Do List application** built using JavaScript to help manage daily tasks efficiently.

---

## 🚀 Features

* ➕ Add new tasks
* ❌ Delete tasks
* ✅ Mark tasks as completed
* 💾 Data stored in browser (Local Storage)
* 📱 Responsive and easy-to-use interface

---

## 🛠️ Technologies Used

* HTML
* CSS
* JavaScript (Vanilla JS)

---

## 📂 Project Structure

```
to-do-app/
│── index.html
│── style.css
│── script.js
```

---

## ⚙️ How It Works

1. User enters a task in the input field
2. Clicks the **Add** button
3. Task gets added to the list
4. Tasks can be marked as completed or deleted
5. All tasks are saved in **local storage**, so they persist after refresh
